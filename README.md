3️⃣ What is the difference between map(), filter(), and forEach()?
ans .
map :It uses each element of an array to create a new element and returns a new array.

filter : It checks each element with a condition and returns a new array with the elements that match the condition.

forEach : It runs a function for every element in the array, but it does not return a new value or array.
